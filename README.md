# BodyStudio

AI-driven diagnostic imaging and biomarker analysis platform for BodyObservability.

## Overview

BodyStudio integrates clinical-grade imaging, sensor fusion, and AI reasoning to provide unified biomarker analysis.

## Hardware Stack

- **Retinal Imaging:** Zeiss Cirrus 6000 OCT/OCTA
- **Body Imaging:** Canon R5 Mark II + macro lens
- **Video Capture:** Canon R5C 4K live feed
- **Depth Sensing:** OAK-D time-of-flight cameras
- **Body Composition:** DEXA scanning
- **Performance Testing:** VO2 max + strength assessment

## Workstation

- 256GB RAM
- NVIDIA Blackwell RTX 6000 Pro (96GB VRAM)
- 4K/8K capture pipeline
